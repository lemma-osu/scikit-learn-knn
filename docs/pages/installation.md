# Installation

!!! info
    `sknnr` will be available through PyPI and conda-forge once it is ready for release. Until then, you can install it from source.

## From source

```bash
pip install git+https://github.com/lemma-osu/scikit-learn-knn-regression@main
```

## Dependencies

- Python >= 3.8
- scikit-learn >= 1.2
- numpy
- scipy
